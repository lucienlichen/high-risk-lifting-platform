### 2.2 技术选型

| 层次 | 技术 | 说明 |
|------|------|------|
| **前端** | Vue 3 + TypeScript + Vite | 沿用现有，无需迁移 |
| **UI 框架** | Element Plus | 沿用现有 |
| **图表** | ECharts 5 | 沿用现有 |
| **HTTP 客户端** | Axios | 新增，替换 Mock |
| **实时通信** | WebSocket (原生 / socket.io-client) | 新增 |
| **后端框架** | Spring Boot 3.2 + Java 17 | 企业级首选，生态成熟 |
| **ORM** | MyBatis Plus 3.5 | 灵活 SQL + 代码生成 |
| **认证授权** | Spring Security + JWT | 无状态 Token 方案 |
| **主数据库** | PostgreSQL 16 | 关系型数据 |
| **时序数据库** | TimescaleDB (PG 扩展) | 监测数据时序存储 |
| **缓存** | Redis 7 | 会话、热数据、实时指标 |
| **消息队列** | Kafka / MQTT Broker | IoT 数据接入 |
| **文件存储** | MinIO / 阿里云 OSS | 报告文件、设备图片 |
| **API 文档** | SpringDoc (OpenAPI 3) | Swagger 自动生成 |
| **部署** | Docker + Docker Compose | 容器化部署 |
